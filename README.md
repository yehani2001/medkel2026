# MedKel 2026 Profile Frame Website

This is a free, no-watermark, static website for creating MedKel 2026 framed profile pictures.

## Files
- `index.html` — the website
- `frame.png` — the transparent MedKel 2026 frame

## How to publish free

### Option 1: Netlify Drop
1. Go to Netlify.
2. Drag and drop this folder or the ZIP contents.
3. Netlify gives you a public link.
4. Share that link with students, faculty, and alumni.

### Option 2: GitHub Pages
1. Create a new GitHub repository.
2. Upload `index.html` and `frame.png`.
3. Go to Settings > Pages.
4. Deploy from the main branch.
5. Share the GitHub Pages link.

### Option 3: Vercel
1. Create a new Vercel project.
2. Import the GitHub repository or drag/drop the folder.
3. Deploy and share the public link.

## Important Facebook note
Facebook usually does not allow an outside website to automatically change a user's profile picture. Users should download the generated image and then set it as their Facebook profile picture manually.
